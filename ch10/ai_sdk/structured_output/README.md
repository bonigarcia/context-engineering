# Structured output with the AI SDK

This example uses `generateText()` with `Output.object()` and a mock model that returns JSON text locally.

## Requirements

* [Node.js](https://nodejs.org/)

## Steps for running this example in the shell

1. Install dependencies:
```bash
npm install
```

2. Run the script:
```bash
npm start
```

## Output

The script prints a validated JavaScript object.
