# Basic text generation with the AI SDK

This example calls `generateText()` from the Vercel AI SDK with `MockLanguageModelV3` from `ai/test`, so it runs locally without any API key.

## Requirements

* [Node.js](https://nodejs.org/)

## Steps for running this example in the shell

1. Install dependencies:
```bash
npm install
```

2. Run the script:
```bash
npm start
```

## Output

The script prints the generated text from the mock model.
