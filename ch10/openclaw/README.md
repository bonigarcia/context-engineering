# OpenClaw

This chapter treats OpenClaw as a local agent control plane for running tasks with explicit tools, approvals, and shared context.

## Examples

- `autonomous_loop/`: how a local agent iterates on a task with bounded steps
- `tool_gatekeeping/`: how tool access stays explicit and constrained
- `human_review_queue/`: how work pauses for manual review before it continues
- `context_sync/`: how the agent keeps workspace state aligned with the task
