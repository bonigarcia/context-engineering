# Dify

This chapter shows Dify as a visual workflow platform for building LLM apps with explicit retrieval, tool use, branching, and output checks.

## Examples

- `context_aware_support/`: a workflow sketch for answering with the right slice of user, conversation, and knowledge context
- `rag_workflow/`: how retrieval feeds grounded answers through a visual pipeline
- `tool_routing/`: how a workflow routes requests to the right external tool
- `guarded_output/`: how guardrails keep the final response within expected bounds
