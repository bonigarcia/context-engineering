# Zapier integration examples

This folder contains hosted trigger-action workflow configurations for Zapier.

## Requirements

- A [Zapier](https://zapier.com/) account

## Examples

- `support_routing/`: Triage customer support emails and alert the right channels.
- `lead_qualification/`: Qualification workflow parsing and scoring leads.
- `ops_notifications/`: Converting system alerts into structured notifications.
- `approval_chain/`: Routing requests to multiple approvers prior to action.

## Running the examples

Each example is in its own folder and contains a `README.md` with instructions on how to run it.
