# Zapier

This chapter shows Zapier as hosted trigger-action automation around AI steps.

## Examples

- `support_routing/`: a support intake flow that classifies a request and sends it to the right team
- `lead_qualification/`: a lead triage flow that scores and routes prospects for follow-up
- `ops_notifications/`: an operations alert flow that turns AI interpretation into the right notification path
- `approval_chain/`: an approval flow that routes work through the right reviewers before action
