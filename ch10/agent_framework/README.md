# Microsoft Agent Framework examples

This folder contains examples demonstrating how to build agents using the Microsoft Agent Framework.

## Requirements

- [Python](https://www.python.org/) 3.10+
- An [OpenAI API key](https://platform.openai.com/api-keys) set as an environment variable (`OPENAI_API_KEY`)

## Examples

- `basic_conversation/`: Basic conversation setup with an agent.
- `agent_with_tool/`: Equipping the agent with custom tools.
- `agent_with_memory/`: Implementing conversational memory in the agent.
- `agent_with_rag/`: Using Retrieval-Augmented Generation (RAG) with the agent.
- `agent_conversational_state/`: Managing state in ongoing conversations.
- `workflow_state/`: Controlling state within multi-step workflows.

## Running the examples

Each example is in its own folder and contains a `README.md` with instructions on how to run it.
