# Ticket triage

This example shows a small Pydantic AI agent that triages support tickets with typed structured output and a local `TestModel` override.

## Requirements

* [Python](https://www.python.org/) 3.10+

## Steps for running this example in the shell

1. Install dependencies:
```bash
python -m venv .venv

# macOS/Linux:
source .venv/bin/activate

# Windows Command Prompt:
.venv\Scripts\activate.bat

# Windows PowerShell:
.venv\Scripts\Activate.ps1

pip install -r requirements.txt
```

2. Run the script:
```bash
python ticket_triage.py
```

## Output

When you run the script, it will print the results to the terminal.
