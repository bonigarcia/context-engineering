# Ticket triage

This example shows a small Pydantic AI agent that triages support tickets with typed structured output and a local `TestModel` override.

## Requirements

- Python 3.10+
- Dependencies in `requirements.txt`

## Run

```bash
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python ticket_triage.py
```
