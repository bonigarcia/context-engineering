# Agent Development Kit (ADK) examples

This folder contains examples demonstrating how to build agents using the Agent Development Kit (ADK).

## Requirements

- [Python](https://www.python.org/) 3.10+
- A [Google API key](https://ai.google.dev/) set as an environment variable (`GOOGLE_API_KEY`)

## Examples

- `basic_agent/`: Basic agent configuration and interaction.
- `agent_with_tool/`: Running an agent equipped with custom tools.
- `context_compression/`: Using context compression to optimize tokens.
- `google_search/`: Integrating Google Search tool for grounding.
- `memory/`: Managing session and user memory in agents.
- `state/`: Managing conversation and workflow state.

## Running the examples

Each example is in its own folder and contains a `README.md` with instructions on how to run it.
