# Flowise

This chapter bucket shows Flowise as a visual builder for context-aware assistants and orchestration patterns.

## Examples

- `context_aware_assistant/`: a chatflow that combines memory, retrieval, and response generation
- `multi_agent_flow/`: how multiple agents pass context through a visual handoff
- `evaluation_tracing/`: how traces and evaluation help inspect flow behavior
- `hitl_control/`: how a human approval step keeps the flow under control
