# Context engineering radar

This example demonstrates how to design and build an interactive, web-based *technology radar* dashboard to catalog and visualize context engineering publications and trends.

The implementation files for this radar are maintained inside the repository's `docs` folder which is served via GitHub Pages:

- Live URL: https://bonigarcia.dev/context-engineering/context-engineering-radar.html
- Sources files:
  - Structure (HTML): [docs/context-engineering-radar.html](/docs/context-engineering-radar.html)
  - Styles (CSS): [docs/css/radar.css](/docs/css/radar.css)
  - Interactive logic (JavaScript): [docs/js/radar-page.js](/docs/js/radar-page.js)
  - Data (JSON): [docs/radar-data.json](/docs/radar-data.json)
