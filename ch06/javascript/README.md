# Prompting examples in JavaScript

This folder contains JavaScript examples for few-shot prompting and prompt chaining.

## Requirements

- [Node.js](https://nodejs.org/)
- An [OpenAI API key](https://platform.openai.com/api-keys)

## Examples

- `few-shot-ticket-normalizer/`: Normalize an unstructured bug report into a support ticket schema.
- `prompt-chaining-support-reply/`: Extract structured context first, then draft a support reply.

## Running the examples

Each example is in its own folder and contains a `README.md` with instructions on how to run it.
