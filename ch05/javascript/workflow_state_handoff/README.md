# Workflow state handoff

This example demonstrates shared workflow state in a two-step JavaScript workflow.

## Requirements

- Node.js 18+
- An OpenAI API key in `OPENAI_API_KEY`

## Install

```bash
npm install
```

## Run

```bash
npm start
```

## Commands

- `/help` - show commands
- `/state` - print the shared workflow state
- `/reset` - clear the workflow state file
- `/exit` - quit
