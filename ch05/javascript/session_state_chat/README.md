# Session state chat

This example demonstrates a structured, transient session state in JavaScript.

## Requirements

- Node.js 18+
- An OpenAI API key in `OPENAI_API_KEY`

## Install

```bash
npm install
```

## Run

```bash
npm start
```

## Commands

- `/help` - show commands
- `/state` - print the current session state
- `/reset` - clear the session state
- `/exit` - quit
